About GameVerse AI

A production-minded Flask application for game discovery, compatibility checks, FPS summaries, recommendation APIs, and streaming AI chat.

How It Works

Game data is loaded from Game_Database.xlsx through Pandas. The chatbot sends each conversation to a real OpenAI-compatible LLM endpoint with recent message history and relevant database context.

Deployment

The app is ready for local Flask execution and Render deployment with Gunicorn.
