About GameVerse AI

A production-minded Flask application for game discovery, compatibility checks, FPS summaries, recommendation APIs, and hybrid AI chat.

How It Works

Game data is loaded from Game_Database.xlsx through Pandas. The chatbot answers factual game questions from that dataset first and uses Groq for reasoning tasks such as recommendations, comparisons, and explanations.

Deployment

The app is ready for local Flask execution and Render deployment with Gunicorn.
