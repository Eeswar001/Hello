# GameVerse AI

GameVerse AI is a Flask web application for game discovery, FPS summaries, PC compatibility checks, JSON APIs, and a streaming LLM chatbot.

## Features

- Responsive dark glassmorphism UI with Bootstrap 5
- Excel-backed game database loaded with Pandas
- Search by name, genre, developer, publisher, platform, and release year
- Filters for genre, platform, developer, and game mode
- Sorting by Steam rating, Metacritic, alphabetical order, newest, and oldest
- Game detail pages with story, requirements, ratings, platforms, and FPS
- PC compatibility checks using CPU/GPU ranking dictionaries
- Streaming chatbot backed by a real OpenAI-compatible chat-completions API
- JSON APIs for chat, search, game details, recommendations, comparison, and compatibility
- Render deployment configuration

## Local Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python app.py
```

Open `http://127.0.0.1:5000`.

## AI Setup

Add your key to `.env`:

```bash
LLM_API_KEY=your_key_here
LLM_MODEL=llama-3.3-70b-versatile
LLM_BASE_URL=https://api.groq.com/openai/v1
```

Groq is the default OpenAI-compatible provider. You can point `LLM_BASE_URL`, `LLM_MODEL`, and `LLM_API_KEY` at another production chat-completions provider such as OpenRouter, Together AI, DeepSeek, Mistral, or Cerebras. Chat responses are generated by the configured LLM with the active conversation history and database context.

## Render Deployment

1. Push this folder to GitHub.
2. Create a new Render web service from the repository.
3. Render will use `render.yaml`.
4. Confirm the Python version is `3.11.9`.
5. Set `LLM_API_KEY` in Render environment variables.

Start command:

```bash
gunicorn app:app
```

## API

- `POST /api/chat` with `{ "message": "Tell me about Elden Ring", "session_id": "browser-session-id" }`
- `POST /api/chat/stream` streams Server-Sent Events for the same payload
- `GET /api/search?q=rpg&sort=metacritic`
- `GET /api/game/elden-ring`
- `POST /api/recommend` with `{ "preferences": "open world rpg", "limit": 5 }`
- `POST /api/compare` with `{ "games": ["Elden Ring", "Baldur's Gate 3"] }`
- `POST /api/compatibility` with `{ "game": "Cyberpunk 2077", "processor": "Intel i7", "graphics_card": "RTX 3060", "ram": 16, "storage": 100 }`
